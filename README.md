# Flask Project

1. Create a new GitHub repository.
2. Clone the repository to your local machine using SSH (generate an SSH key if needed, add the public key to your GitHub account).
3. Create a new branch named after your username (e.g., Tutedude).
4. Add your Flask project files to this branch.
5. Commit the changes and merge the branch into the main branch.

Run commands:

pip install -r requirements.txt

python app.py
